# Documentation

## Used stack

Backend:

- Node v22.13.1
- Fastify

Frontend:

- React
- Vite

Open API OpenWeather

## Installation

Requires an env file in the backend folder

Backend:

Backend can be run through Docker

```
docker compose up --build --renew-anon-volumes -d
```

Server starts on http://localhost:3000

Frontend:

```
cd frontend
npm install
npm run dev
```

Vite starts on http://localhost:5173/
